# TELECOM-CUSTOMER-CHURN-ANALYSIS-PROJECT
Analyzed telecom customer churn (27% churn rate) using SQL Server for data cleaning/ETL, Python (Random Forest, 84% accuracy) for predicting at-risk customers, and Power BI for an interactive 2-page dashboard covering churn drivers and a future-churn-risk view. Tools: SQL Server, Power BI, Python (scikit-learn), Excel.
